问题

给定kkk维空间的KKK个点集(在几何距离上相近)。每次查询一个新点，问这个点最有可能属于哪一个点集。

kkk在一定范围内增大不会影响我们的解法，因此这里我们先从简处理k=2k = 2k=2的情况，即222维平面的点集。


方案

如何表示一个划分方案？

用一个K×1K \times 1K×1向量表示，第iii个数字表示这个点属于点集iii的概率。

对于初始点集作为给出的标准答案来说一个属于iii点集的点属于点集iii的概率当然是111。


损失函数

如何评价一个划分方案的好坏？

在监督学习(提供标准答案)的情况下，常用的有两类评价函数，一种是平方差损失函数(MSEMSEMSE)，公式如下:

Loss=12K∑i=1K(yi−y^i)2Loss = \frac{1}{2K}\sum_{i=1}^K (y_i - \hat y_i)^2 Loss=2K1​i=1∑K​(yi​−y^​i​)2

这个1/21/21/2是为了求导方便。MSEMSEMSE是一种很容易想到而且有效性显而易见的损失函数。不过另一种相对更好的评价函数是交叉熵损失函数(CECECE)，公式如下:

Loss=−∑i=1Kyiln⁡(y^i)Loss = -\sum_{i=1}^Ky_i\ln(\hat y_i) Loss=−i=1∑K​yi​ln(y^​i​)

注意这里∑i=1Kyi=1\sum_{i=1}^Ky_i = 1∑i=1K​yi​=1。LossLossLoss越小的划分方案我们认为越优，直观理解就是对于yiy_iyi​比较大的项，追求ln⁡(1y^i)\ln(\frac{1}{\hat y_i})ln(y^​i​1​)较小即1y^i\frac{1}{\hat y_i}y^​i​1​较小即y^i\hat y_iy^​i​较大，这与yiy_iyi​是一致的。

约定和初始化

为了处理好细节问题，现在这里对神经网络的结构做一下说明和实现。

一开始我试图用一个二维数组来表示神经网络，每个神经元对应一个坐标(i,j)(i,j)(i,j)。这种方法因为内存连续性，常数比较优秀。但是不能很好的处理每层节点数不同的情况，无法自定义空间复杂度已经最大层数和最大节点数。

还有一种常见的方法使用矩阵表示所有的转移。这样的实现简洁不易出错，代码更易理解。但是灵活性较差，往往需要使用第三方库(例如OpenCV)。而手动实现矩阵则凭空增加了代码复杂度。

于是我选择用vector和struct组合来完成这一任务。这种做法是最直观的(相对于矩阵方法更贴近于神经网络的物理意义而不是数学意义)。


定义一个神经网络有N+1N+1N+1层，分别记为第0,1,2,...,N0,1,2,...,N0,1,2,...,N层。

其中第000层为输入层，只有k=2k = 2k=2个节点，输入形式为x/RANGE,y/RANGEx/\text{RANGE},y/\text{RANGE}x/RANGE,y/RANGE，其中xxx为坐标范围，这样保证输入的每个数都与111在一个数量级上，防止sigmoid\text{sigmoid}sigmoid函数将输入"抹平"。

所有000到N−1N-1N−1层的激活函数都是sigmoid\text{sigmoid}sigmoid，但第NNN层的激活函数为Softmax\text{Softmax}Softmax，作为输出层，有恰好KKK个节点。

第iii层有L[i].n+1L[i].n+1L[i].n+1个节点，其中000节点代表偏置，因此000节点的输出总是111，偏置由边权调整。以后用(i,j)(i,j)(i,j)表示第iii层第jjj个节点。

现在每个www都是一开始设定好的，这个神经网络根本没有学习能力。如何让神经网络具有学习能力？

关键就是通过大量数据调整所有wi,j,kw_{i,j,k}wi,j,k​的值，使得最后的LossLossLoss最小。

一个很常见的思路就是求导: 每次贪心地将每个wi,j,kw_{i,j,k}wi,j,k​尝试调大或者调小。至于具体调大还是调小，当然就是向着改变以后LossLossLoss会降低的方向调整。LossLossLoss降低的方向就是LossLossLoss关于wi,j,kw_{i,j,k}wi,j,k​的偏导数正向的反方向。即每次调整:

wi,j,k=wi,j,k−η∂Loss∂wi,j,kw_{i,j,k} = w_{i,j,k} - \eta \frac{\partial Loss}{\partial w_{i,j,k}} wi,j,k​=wi,j,k​−η∂wi,j,k​∂Loss​

这种方法就是梯度下降算法。η\etaη是学习率(就是下降的步长)。

先来尝试推导最后一层节点的偏导数。首先回忆之前的交叉熵损失函数以及Softmax\text{Softmax}Softmax函数:

Loss=−∑i=1KAnsiln⁡(Out^i)Outj=ehN,j∑iehN,i=ehN,jSumLoss = - \sum_{i=1}^KAns_i\ln(\hat{Out}_i) \\ Out_j = \frac{e^{h_{N,j}}}{\sum_i e^{h_{N,i}}} = \frac{e^{h_{N,j}}}{Sum} \\ Loss=−i=1∑K​Ansi​ln(Out^i​)Outj​=∑i​ehN,i​ehN,j​​=SumehN,j​​

∂Outi∂hN,j=−ehN,iehN,jSum2=−OutiOutj∂Outj∂hN,j=Sumi≠jehN,jSum2=(1−Outj)Outj\frac{\partial Out_i}{\partial h_{N,j}} = -\frac{e^{h_{N,i}}e^{h_{N,j}}}{Sum^2} = -Out_iOut_j \\ \frac{\partial Out_j}{\partial h_{N,j}} = \frac{Sum_{i \ne j} e^{h_{N,j}}}{Sum^2} = (1-Out_j)Out_j \\ ∂hN,j​∂Outi​​=−Sum2ehN,i​ehN,j​​=−Outi​Outj​∂hN,j​∂Outj​​=Sum2Sumi​=j​ehN,j​​=(1−Outj​)Outj​

∂Loss∂hN,j=−∑i=1KAnsi1Outi∂Outi∂hN,j=−Ansj(1−Outj)+∑i≠jAnsiOutj=−Ansj+∑i=1KAnsiOutj=Outj−Ansj\frac{\partial Loss}{\partial h_{N,j}} = -\sum_{i=1}^K Ans_i\frac{1}{Out_i}\frac{\partial Out_i}{\partial h_{N,j}} = -Ans_j(1-Out_j) + \sum_{i \ne j} Ans_iOut_j \\ = -Ans_j + \sum_{i=1}^K Ans_iOut_j = Out_j - Ans_j ∂hN,j​∂Loss​=−i=1∑K​Ansi​Outi​1​∂hN,j​∂Outi​​=−Ansj​(1−Outj​)+i​=j∑​Ansi​Outj​=−Ansj​+i=1∑K​Ansi​Outj​=Outj​−Ansj​

最后一个等式是因为AnsAnsAns是一个概率分布，∑i=1KAnsi=1\sum_{i=1}^K Ans_i = 1∑i=1K​Ansi​=1。于是就有:
