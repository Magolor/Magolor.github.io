给定一个收敛数列，近似估计这个数列的收敛值至所要求的的精度。例如:

1−13+15−17+19−...=π41 - \frac{1}{3} + \frac{1}{5} - \frac{1}{7} + \frac{1}{9} - ... = \frac{\pi}{4} 1−31​+51​−71​+91​−...=4π​

有什么办法可以快速求这个收敛值的近似值呢？

Richardson外推法(Richardson Extrapolation)

Richardson外推法是另外一种加速级数求和的方式。Shanks变换一般对于交错数列(AiA_iAi​每项一正一负)效果更好，而Richardson外推法对于单调数列效果更好。

Richardson外推法思路和Shanks变换几乎完全一样。不过，Shanks变换构造sn(B)=S+αqns_n(B) = S + \alpha q^nsn​(B)=S+αqn，而Richardson外推法则构造:

sn(B)=S+∑i=1+∞Ainis_n(B) = S + \sum_{i=1}^{+\infty}\frac{A_i}{n^i} sn​(B)=S+i=1∑+∞​niAi​​

先只计算第一项，联立方程组:

{sn+1(B)=S+A1n+1sn(B)=S+A1n\begin{cases} s_{n+1}(B) = S + \frac{A_1}{n+1} \\ s_n(B) = S + \frac{A_1}{n} \\ \end{cases} \\ {sn+1​(B)=S+n+1A1​​sn​(B)=S+nA1​​​

用n+1n+1n+1倍的①式减去nnn倍的②式，得到的结果:

S=(n+1)sn+1(B)−nsn(B)S = (n+1)s_{n+1}(B) - ns_n(B) S=(n+1)sn+1​(B)−nsn​(B)

同样地，目前为止严格精确。因为同样满足lim⁡n→+∞sn(B)=S\lim_{n \rightarrow +\infty}s_n(B) = Slimn→+∞​sn​(B)=S，所以同样地，Richardson外推法假设sn(B)s_n(B)sn​(B)可以近似地等于sn(A)s_n(A)sn​(A)。

总结

Shanks变换和Richardson外推法都是通过构造一个带有附加项的近似序列来解决收敛级数求和的问题。

Shanks变换更适用于交错数列，Richardson外推法更适用于单调数列。

注意两者都只用到了数列的某几项的信息。这意味着这两种方法不依赖于具体的数列/函数。同样这也意味着两种方法存在不足: 无法预测存在"突变"的函数。
